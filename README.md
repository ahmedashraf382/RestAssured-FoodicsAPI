# RestAssured-FoodicsAPI

# Installation
To install and run the project, follow these steps:

Clone the repository using git clone https://github.com/ahmedashraf382/RestAssured-FoodicsAPI.git
Navigate to the cloned directory using

"cd RestAssured-FoodicsAPI"

Install the project dependencies using

"mvn install"

# Running the tests
To run the tests, use the following command:

"mvn test"

This will run all of the tests in the project and output the results to the console.

Note: Before running the tests, make sure to update the "config.properties" file with your own Foodics API credentials. You can obtain these credentials by signing up for a Foodics account and generating an API key.
